## Задание

У вас есть данные

```
let crypto = [
  {
    name : "Bitcoin",
    price: 1388.37
  },
  {
    name : "Ethereum",
    price: 716
  },
  {
    name : "Litecoin",
    price: 140
  }
]
```

На их основе создайте страницу, демонстрирующую сравнение курсов криптовалюты
— Заголовок берем из name
— Цену пишем в параграф
— Устанавливаем на странице цветной блок, ширина которого привязана к цене валюты
